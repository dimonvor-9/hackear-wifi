| Opción | Función | Descripción |

| **[3]** | **Crear lista de palabras personalizada** | Genera diccionarios personalizados basados en palabras clave, fechas o patrones específicos |
| **[4]** | **Use default wordlist predeterminada** | Utiliza el diccionario integrado con más de 50 millones de contraseñas comunes |

| **[6]** | **Find password** | Inicia el proceso de recuperación de contraseña contra el handshake capturado |

## Cómo Instalar 📥

### Instalación Estándar (Windows)

1. Descarga `HackearWiFi.exe` desde el enlace oficial de GitHub
2. Ejecuta el archivo (no requiere instalación)
3. Selecciona opción **[1]** para escanear redes disponibles
4. Identifica la red objetivo y captura su handshake con opción **[2]**
5. Opcional: Crea lista personalizada con opción **[3]** o usa diccionario predeterminado con opción **[4]**
6. Selecciona idioma con opción **[5]** si es necesario
7. Ejecuta opción **[6]** para encontrar la contraseña
8. Espera los resultados del análisis

## Preguntas Frecuentes ❓

**Q: ¿HACKEAR WIFI es realmente gratis?**
A: Sí. Es completamente gratuita, creada por dimonvor-9 para la comunidad.

**Q: ¿Qué significa EFILITE?**
A: Es el nombre original del proyecto, ahora llamado HACKEAR WIFI para mayor claridad.

**Q: ¿Cómo uso la opción manual manual?**
A: Es para captura avanzada de handshakes. Permite seleccionar canales específicos y ajustar parámetros.

**Q: ¿Puedo crear mi propia lista de palabras?**
A: Sí. La opción [3] te permite generar diccionarios personalizados con palabras clave.

**Q: ¿La wordlist predeterminada es suficiente?**
A: Incluye millones de contraseñas comunes. Para contraseñas complejas, usa listas personalizadas.

**Q: ¿Necesito adaptador especial?**
A: Para escaneo básico funciona cualquier adaptador. Para captura de handshake se recomienda adaptador con modo monitor.

**Q: ¿Hay riesgo de malware?**
A: La herramienta es limpia. Algunos antivirus pueden marcarla por su naturaleza, pero es un falso positivo.

**Q: ¿Cómo cambio el idioma?**
A: Usa la opción [5] y selecciona Español o English.

## Solución de Problemas 🔧

| Problema | Solución |
|----------|----------|
| **"Ejecutar como Administrador"** | Haz clic derecho y selecciona "Ejecutar como administrador" |
| **No detecta redes** | Verifica que el adaptador WiFi esté activo y compatible |
| **Captura de handshake falla** | Usa opción manual, espera conexión activa, ejecuta desautenticación |
| **Diccionario no carga** | Verifica espacio en disco, reinicia herramienta |
| **Idioma no cambia** | Reinicia la herramienta después de seleccionar idioma |
| **Windows Defender bloquea** | Agrega carpeta a exclusiones—herramientas de seguridad son marcadas |
